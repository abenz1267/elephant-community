# Background Selector
Implementation from the provided [example](https://benz.gitbook.io/walker/customization/custom-menus#dynamic-menu-example) in the [gitbook](https://benz.gitbook.io/walker)
- Choose your background directly using this custom menu.
- Solution if you want to limit installed applications in your machine
- Can cover multiple directories if you want to organize your wallpapers in different folders. Here's the [example](https://github.com/dhms013/omarchy/blob/844371d7a65db94fc204e03ab434c140901af7f7/default/elephant/omarchy_background_selector.lua)

# Requirements:
- awww/swaybg/swww/anything that you prefer
- Replace the [wallpaper_dir](https://github.com/dhms013/elephant-community/blob/30f500eaa00d140f8832b5fa25aa50d18e0e6518/background-selector/backgroundSelector.lua#L9) value with your real path
- Replace the [activate](https://github.com/dhms013/elephant-community/blob/30f500eaa00d140f8832b5fa25aa50d18e0e6518/background-selector/backgroundSelector.lua#L24) command with your preferred wallpaper tool
